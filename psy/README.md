# Личностен Психотест | BodyBest

## Описание

Standalone HTML психологически тест, който оценява личностни характеристики и предоставя AI анализ на потенциални връзки с хранителните навици.

## Файлове

- **psychotest.html** - Основният standalone HTML файл с въпросника, логика за изчисляване и AI анализ
- **bb_optional_personality_v1_1.json** - Въпросникът с 14 въпроса
- **bb_optional_personality_key_v1_1.json** - Ключ за оценяване и интерпретация на резултатите
- **index.html** - Визуален тест (съществуващ)
- **01.jpg - 08.jpg** - Изображения за визуалния тест

## Как работи psychotest.html

### 1. Структура на теста

Тестът съдържа 14 въпроса, всеки с 3 равностойни опции. Въпросите оценяват 7 личностни измерения:

- **E** (Extraversion) - Социална енергия и общителност
- **C** (Conscientiousness) - Структура и самодисциплина
- **O** (Openness) - Отвореност към новото
- **A** (Agreeableness) - Мекота в контакт
- **N** (Neuroticism) - Емоционална реактивност
- **I** (Impulsivity) - Импулсивност
- **R** (Risk-seeking) - Рискова ориентация

### 2. Изчисляване на резултати

Системата използва scoring key от JSON файла:
1. За всеки въпрос се взема теглото на избраната опция
2. Теглата се сумират за всяка дименсия (raw score)
3. Изчислява се min/max възможен score за всяка дименсия
4. Резултатите се нормализират към скала 0-100

### 3. Type Code

Генерира се 4-буквен комуникационен код (напр. E-V-M-J):
- Първа буква: E (общителен) vs X (затворен)
- Втора буква: V (новаторски) vs S (практичен)
- Трета буква: M (мек) vs D (директен)
- Четвърта буква: J (структуриран) vs P (гъвкав)

### 4. AI Анализ на хранителни навици

Системата анализира комбинацията от личностни черти и идентифицира:

#### Потенциални рискове:
- Рестриктивни тенденции (високо C + високо N)
- Емоционално хранене (високо N + ниско C)
- Импулсивно хранене (високо I)
- Хаотични навици (ниско C + високо O)
- Екстремни подходи (високо R + високо I)
- Социално хранене (високо E + ниско A)
- Ограничаващ режим (ниско O + високо C)

#### Силни страни:
- Добра самодисциплина (C >= 60)
- Отвореност към ново (O >= 60)
- Ниска емоционална реактивност (N < 50)
- Ниска импулсивност (I < 50)
- Предпазлив подход (R < 50)

#### Конкретни препоръки:
- Специфични стратегии според личностния профил
- Техники за управление на рискови навици
- Съвети за използване на силните страни

## Използване

### За крайни потребители:

1. Отворете `psy/psychotest.html` в браузър
2. Прочетете инструкциите
3. Кликнете "Започни теста"
4. Отговорете на всички 14 въпроса
5. Разгледайте резултатите и AI анализа

### За разработчици:

#### Интеграция в проекта:

```html
<!-- Линк към теста -->
<a href="psy/psychotest.html">Личностен тест</a>
```

#### Модификация на въпросите:

Редактирайте `testData` обекта в `<script>` секцията:

```javascript
const testData = {
  "items": [
    {
      "id": "psy_01",
      "text": "Вашият въпрос?",
      "options": ["Опция 1", "Опция 2", "Опция 3"],
      "legend": "Легенда"
    }
  ]
};
```

#### Модификация на scoring:

Редактирайте `scoringKey` обекта:

```javascript
const scoringKey = {
  "psy_01": [
    {"C": 2, "O": -1, "E": 0, ...},  // Опция 0
    {"C": 1, "O": 0, "E": 0, ...},   // Опция 1
    {"C": -1, "O": 1, "E": 1, ...}   // Опция 2
  ]
};
```

#### Добавяне на нови правила за AI анализ:

В функцията `analyzeNutritionalRisks()`:

```javascript
// Пример: Нов риск
if (scores.DIMENSION >= threshold) {
  risks.push({
    title: 'Заглавие на риска',
    description: 'Описание на риска'
  });
  recommendations.push('Препоръка за действие');
}
```
